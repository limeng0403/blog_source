# 个人博客源代码

此博客使用hexo开源框架制作。

可以先到[hexo](https://hexo.io/zh-cn/)查看一下文档，有中文。

此源代码不包含各种包，如果需要，请自行安装。

### tips:

- 安装nodejs：[https://nodejs.org](https://nodejs.org/en/)

```bash
# npm install
# npm install hexo -g
# hexo clean
# hexo generate
# hexo server -p 3001
```

- 使用浏览器打开：127.0.0.1:3001
- 请不要使用`hexo deploy`指令，请不要恶意向服务器上传恶意文件。谢谢合作。
